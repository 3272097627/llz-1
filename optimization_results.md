# 🎯 反应速率提升总结

## 问题与解决

### 问题：出口处占比全是 0%
- CaO, C2S, C3S, C3A, C4AF (熟料): 0.00%
- 原因：物料无法流动 + 反应速率太低

### 解决方案：
1. ✅ **修复物料流动**：在每个时间步开始时进行单元间转移
2. ✅ **提升反应条件**：增加温度、反应速率系数、停留时间

## 📊 优化结果对比

| 指标 | 优化前 | 优化后 | 变化 |
|------|-------|--------|------|
| CaO 占比 | 0% | **54.23%** | ↑↑↑ |
| CaCO3 占比 | 77.60% | **23.37%** | ↓ |
| C2S 占比 | 0% | 0% | 需进一步优化 |
| 出口温度 | 300K | 2000K | 异常(需修复) |

## 🔧 关键参数调整

### 1. 温度参数
```python
# 固体进料温度
solid_feed['temperature'] = 600  # K (↑ 从 390K)

# 气体进料温度  
gas_feed['temperature'] = 1300  # K (↑ 从 800K)
```
**效果**：温度是反应速率的指数，每增加 100K 可显著加快反应

### 2. 反应速率系数 (kr)
```python
'r1': 1e8   # CaCO3 分解 (保持不变)
'r2': 1e9   # C2S 形成 (↑ 从 1e7，增加 100 倍)
'r3': 1e10  # C3S 形成 (↑ 从 1e9，增加 10 倍)
'r4': 1e9   # C3A 形成 (↑ 从 1e8，增加 10 倍)
'r5': 1e9   # C4AF 形成 (↑ 从 1e8，增加 10 倍)
```
**效果**：直接增加反应速率，对复杂熟料反应影响最大

### 3. 时间步长 (dt)
```python
dt = 0.08  # (↑ 从 0.01，增加 8 倍)
```
**效果**：每个单元停留时间更长，反应进行更充分

## 💡 反应进展

### 第一阶段：CaCO3 分解 ✅
```
CaCO3 → CaO + CO2
```
- 所需温度：400-900K
- **当前结果**：CaCO3 从 77.60% → 23.37%，分解率 70%

### 第二阶段：CaO 与 SiO2 反应 ✅
```
2CaO + SiO2 → C2S (二钙硅酸盐)
```
- 所需温度：1200-1450K
- **当前结果**：C2S 占比 0%（需进一步提升温度或反应系数）

### 第三阶段：C2S → C3S 反应 ⏳

CaO + C2S → C3S (三钙硅酸盐)

- **当前结果**：C3S 占比 0%（需要 C2S 先生成）

## ⚠️ 注意事项

1. **出口温度 2000K 异常**：
   - 可能是热功率计算有数值溢出
   - 需要检查温度计算公式

2. **物质守恒检查**：
   - ✅ 当前占比之和 ≈ 100%（正确）
   - 应该定期检查组分浓度总和

3. **数值稳定性**：
   - 如果 dt 过大可能导致不稳定
   - 如果 kr 过大可能导致计算溢出
   - 建议增量调参，逐步优化
